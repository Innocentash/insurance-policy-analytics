# insurance-policy-analytics
Insurance Policy Analytics Project using Python, SQL, and Power BI

## Overview
This project automobile insurance policy data and analyzes claim costs and portfolio profitability.

Tools Used:
- Python (data simulation)
- SQL (data analysis)
- Power BI (dashboard visualization)

## Dataset
The project contains two datasets:
- Policy Sales Data
- Claims Data

Total policies simulated: 1,000,000

## Key Metrics
Total Premium: ₹239.9M  
Total Claims: ₹493.6M  
Loss Ratio: 205.77%

## Dashboard
The Power BI dashboard visualizes:

- Total Premium
- Total Claims
- Loss Ratio
- Claim Trends by Month
- Premium by Policy Tenure

## Key Insight
The portfolio is unprofitable because claims exceed premium revenue.
